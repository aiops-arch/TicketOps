## 🎯 Deploy in 3 Steps (15 minutes)

### Step 1️⃣: Deploy to Vercel (2 minutes)
```
1. Go to → https://vercel.com/new
2. Select → aiops-arch/TicketOps repository
3. Click → Deploy
4. Wait → 1-2 minutes
✅ LIVE URL: https://ticketops.vercel.app
```

### Step 2️⃣: Setup Supabase (5 minutes)
```
1. Go to → https://supabase.com
2. Click → New Project
3. Fill → Project details
4. Copy → API keys (SUPABASE_URL, SUPABASE_KEY)
```

### Step 3️⃣: Configure Environment Variables (3 minutes)
```
In Vercel Project Settings → Environment Variables:
- SUPABASE_URL = <copy from Supabase>
- SUPABASE_KEY = <copy from Supabase>
- JWT_SECRET = <create any random string>

Redeploy → Done!
```

## 🚨 Important Security Notes

1. **Never commit `.env` to GitHub** ⚠️
2. Use environment variables for all secrets
3. Generate strong JWT_SECRET
4. Enable Supabase Row Level Security
5. Rotate credentials regularly
